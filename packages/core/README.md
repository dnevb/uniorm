# Uniorm Core

Core Uniorm package

## Getting Started

```js
import uniorm from "@uniorm/core";
```

### Installing

```bash
pnpm add @uniorm/core
yarn add @uniorm/core
npm install @uniorm/core
```

## Running the tests

```bash
pnpm run test
```

## Built With

- [knex](https://knexjs.org) - Sql query builder
